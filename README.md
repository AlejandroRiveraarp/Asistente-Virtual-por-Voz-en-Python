# Asistente Virtual por Voz en Python 🎤 
- 📌 Descripción

Este proyecto es un asistente virtual controlado por voz desarrollado en Python.
Permite interactuar mediante comandos hablados para realizar tareas como buscar información, reproducir música, contar bromas y más.

Funcionalidades
- ✅ Reconocimiento de voz (Speech-to-Text)
- ✅ Respuesta hablada (Text-to-Speech)
- ✅ Abrir páginas web (YouTube, Google)
- ✅ Consultar día y hora
- ✅ Búsquedas en Wikipedia
- ✅ Búsquedas en Internet
- ✅ Reproducción de música en YouTube
- ✅ Contar bromas
- ✅ Consulta de precios de acciones (Apple, Amazon, Google)

---

Tecnologías utilizadas
- Python 🐍
- speech_recognition
- pyttsx3
- pywhatkit
- wikipedia
- yfinance
- pyjokes

---

🎤 Ejemplos de comandos

Puedes decir cosas como:

- "Abrir YouTube"
- "Abrir navegador Google"
- "¿Qué día es hoy?"
- "¿Qué hora es?"
- "Busca en internet inteligencia artificial"
- "Reproducir música electrónica"
- "Cuéntame una broma"
- "Precio de las acciones de Apple"
- "Adiós"
